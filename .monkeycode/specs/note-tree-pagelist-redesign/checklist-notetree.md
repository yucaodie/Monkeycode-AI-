# NoteTree 交互清单

## 视觉状态
- [ ] 正常态：图标 + 名称 + 笔记计数 badge（始终显示，含 0）
- [ ] 悬停态：浅色背景高亮 + 操作按钮 opacity 淡入（重命名 / 删除）
- [ ] 选中态：深色背景高亮 + 名称主色 + 操作按钮保持可见
- [ ] 编辑态：名称变为输入框，操作按钮隐藏，Enter/blur 提交，Esc 取消
- [ ] 加载态：3 行灰色骨架屏 + shimmer 闪烁动画
- [ ] 空态：引导文案"点击 + 创建笔记本"
- [ ] 错误态：新建/重命名时名称重复 → 内联红色提示"同名已存在"

## 动作
- [ ] 单击未选中项 → 选中 + 加载对应笔记页
- [ ] 单击已选中项 → 进入内联重命名
- [ ] 双击任意项 → 进入内联重命名
- [ ] 重命名提交 → 乐观更新名称（保留 note_count），API 失败回滚
- [ ] 删除按钮 → 弹出自定义确认对话框 → 确认后删除 + 若选中则清除选中
- [ ] 新建按钮 → 列表顶部出现输入框 → Enter 创建 → 插入顶部（note_count=0）
- [ ] 名称重复校验：空值/仅空白/同名 → 不提交

## 过渡动画
- [ ] 背景色切换：150ms ease
- [ ] 操作按钮淡入淡出：100ms opacity
- [ ] 新建输入框：150ms max-height + opacity 滑入

## 键盘导航
- [ ] ↑↓ 移动焦点
- [ ] Enter 选中
- [ ] F2 重命名
- [ ] Delete 弹出删除确认
- [ ] Esc 取消编辑/新建

## 数量实时联动
- [ ] PageList 创建/删除笔记页 → +1 notebooksVersion → NoteTree 自动重载

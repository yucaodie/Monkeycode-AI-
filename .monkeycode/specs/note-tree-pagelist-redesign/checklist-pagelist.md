# PageList 交互清单

## 视觉状态
- [ ] 正常态：图标 + 标题（无计数 badge）
- [ ] 悬停态：浅色背景高亮 + 操作按钮 opacity 淡入（重命名 / 删除）
- [ ] 选中态：深色背景高亮 + 标题加粗 + 操作按钮保持可见
- [ ] 编辑态：标题变为输入框，Enter/blur 提交，Esc 取消
- [ ] 加载态：骨架屏（同 NoteTree）
- [ ] 空态：引导文案"点击 + 创建笔记页"
- [ ] 错误态：同 NoteTree 的内联重复提示

## 动作
- [ ] 单击未选中项 → 选中 + 右侧加载编辑器
- [ ] 单击已选中项 → 进入内联重命名
- [ ] 重命名提交 → 乐观更新标题，API 失败回滚
- [ ] 删除按钮 → 确认对话框 → 删除 + 清除选中 + 触发 refreshNotebooks
- [ ] 新建按钮 → 自动创建笔记（空笔记本时）+ 去重递增标题 → 插入列表 + 触发 refreshNotebooks
- [ ] 标题去重：同 noteId 下"新笔记页"→"新笔记页2"→"新笔记页3"

## 过渡动画
- [ ] 同 NoteTree：背景 150ms + 操作按钮 100ms + 输入框 150ms
